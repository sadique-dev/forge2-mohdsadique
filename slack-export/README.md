# Slack Export

> **Option A (preferred):** Place the full unzipped Slack export in this directory.
> Go to: Slack Workspace Settings → Import/Export → Export → Entire history → Download → unzip here.

> **Option B (fallback):** If export is locked on your Slack plan, place named screenshots in `screenshots/`.

## Required screenshot filenames (Option B)

| File | What it must show |
|---|---|
| `sprint-main-01.png` | You giving Hermes a sprint goal + Hermes posting the plan/backlog back |
| `agent-coder-01.png` | Hermes assigning a scoped issue to OpenClaw |
| `agent-log-01.png` | OpenClaw's structured report (What I Did / What's Left / What Needs Your Call) |
| `ci-cd-01.png` | CI test results for at least one PR |
| `human-review-01.png` | You approving a release candidate before merging |

> ⚠️ All screenshots must show real content with real timestamps. Staged or reconstructed screenshots are a DQ risk.
